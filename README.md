# 💼 Job Search Dashboard (React.js)

A responsive Job Search Dashboard built using React.js with API integration, search/filter, routing, and save job feature using localStorage.

---

## 🚀 Features
- Search jobs
- View job details
- Save jobs
- API integration
- React Router navigation

---

## 🛠️ Tech Stack
- React.js
- Axios
- React Router DOM
- JavaScript

---

## ▶️ Run Project
npm install  
npm run dev
